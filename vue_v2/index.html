# 为什么还学习2.x版本的源码

Vue3.0的源码已经开放，当前处于pre-alpha版本，但2.x版本自发布至今已发展成一个成熟而稳定的框架， 与Angular、React并列三大框架，有着完善的生态，Vue3.0当前处于起步阶段，生态还未建立起来，从2.x过渡到 3.0还需要时间。作为一个成熟稳定的框架，Vue2.x源码有许多值得学习的地方，且目前工作中的项目基本都是使用 2.x版本的Vue，学习2.x版本的源码既可以提高JS内功，又有助于更好的排查日常开发中的遇到的问题。
